# Mereology Pattern

This ontology specifies the foundation-level concepts related to the mereology pattern of the city data model.

This pattern imports the following files:

- [https://w3id.org/citydata/part1/v1/Core/](https://w3id.org/citydata/part1/v1/Core/)

This pattern does not contain any classes.


The formal definition of this pattern is available in [OWL Syntax](MereologyPattern.owl).
