# onDateTimeDescription

Specifies the date-time description (as defined in the Time pattern) on which a recurring event recurs. For example, the day of the month on which a MonthlyRecurringEvent recurs, or the day and month on which a YearlyRecurringEvent recurs.

**Domain**: [RecurringEvent](../classes/RecurringEvent.md)

**Range**: [time:DateTimeDescription](https://w3id.org/citydata/imported/time/DateTimeDescription)

**IRI**: `https://w3id.org/citydata/part1/v1/onDateTimeDescription`
