# hasClaim

Indicates that an Agent is the beneficiary of an Activity fulfilled by another Agent in the Agreement (i.e., the Agent with the duty to fulfil the Activity), e.g. payment of wages, provision of services.

**Range**: [AtomicAgreement](../classes/AtomicAgreement.md)

**IRI**: `https://w3id.org/citydata/part1/v1/hasClaim`
