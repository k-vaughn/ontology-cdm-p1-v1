# hasLatestManifestation

Identifies the most recent Manifestation that is related to a particular FirstManifestation (i.e., for some snapshot of an object, it identifies the most recent snapshot of the object).

**Domain**: [FirstManifestation](../classes/FirstManifestation.md)

**Range**: [Manifestation](../classes/Manifestation.md)

**IRI**: `https://w3id.org/citydata/part1/v1/hasLatestManifestation`
