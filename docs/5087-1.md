# 5087-1

This ontology specifies the foundation-level concepts of the city data model.

This pattern imports the following files:

- [http://www.w3.org/ns/prov#](http://www.w3.org/ns/prov#)
- [https://w3id.org/citydata/part1/v1/AgreementPattern/](https://w3id.org/citydata/part1/v1/AgreementPattern/)
- [https://w3id.org/citydata/part1/v1/OrganizationStructurePattern/](https://w3id.org/citydata/part1/v1/OrganizationStructurePattern/)
- [https://w3id.org/citydata/part1/v1/RecurringEventPattern/](https://w3id.org/citydata/part1/v1/RecurringEventPattern/)

This pattern does not contain any classes.


The formal definition of this pattern is available in [OWL Syntax](5087-1.owl).
