# Information technology - City data model - Part 1: Foundation level concepts - Generic Properties Pattern

This ontology specifies the foundation-level concepts related to the generic properties pattern of the city data model.

This pattern imports the following files:

- [https://w3id.org/citydata/part1/v1/Core](https://w3id.org/citydata/part1/v1/Core)

This pattern does not contain any classes.
This module defines the following properties:

- [GeneralDataProperty](../properties/GeneralDataProperty.md)
- [hasDescription](../properties/hasDescription.md)
- [hasIdentifier](../properties/hasIdentifier.md)
- [hasName](../properties/hasName.md)


The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../generic-properties-pattern.ttl) and the SHACL [restrictions](../generic-properties-shacl.ttl).
