# Information technology - City data model - Part 1: Foundation level concepts - Core

This ontology specifies the foundation-level concepts that are used by all other patterns of the city data model.

This pattern imports the following files:

- [http://www.w3.org/2006/time#](http://www.w3.org/2006/time#)

This pattern consists of the following classes:

- [Cdm Thing](CdmThing.md)
- [Foundational Cdm Thing](FoundationalCdmThing.md)
This module defines the following properties:

- [CdmDataProperty](../properties/CdmDataProperty.md)
- [CdmObjectProperty](../properties/CdmObjectProperty.md)
- [FoundationalCdmDataProperty](../properties/FoundationalCdmDataProperty.md)
- [FoundationalCdmObjectProperty](../properties/FoundationalCdmObjectProperty.md)


The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../core-pattern.ttl) and the SHACL [restrictions](../core-shacl.ttl).
