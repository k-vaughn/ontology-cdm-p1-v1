# Information technology - City data model - Part 1: Foundation level concepts - Resource Pattern

This ontology specifies the foundation-level concepts related to the resource pattern of the city data model.

This pattern imports the following files:

- [https://w3id.org/citydata/part1/v1/ActivityPattern](https://w3id.org/citydata/part1/v1/ActivityPattern)
- [https://w3id.org/citydata/part1/v1/CityUnitsPattern](https://w3id.org/citydata/part1/v1/CityUnitsPattern)

This pattern consists of the following classes:

- [Consume State](ConsumeState.md)
- [Divisible Resource](DivisibleResource.md)
- [Non Divisible Resource](NonDivisibleResource.md)
- [Planned Allocation](PlannedAllocation.md)
- [Produce State](ProduceState.md)
- [Release State](ReleaseState.md)
- [Resource](Resource.md)
- [Resource Thing](ResourceThing.md)
- [Terminal Resource State](TerminalResourceState.md)
- [Use State](UseState.md)
This module defines the following properties:

- [capacityInUse](../properties/capacityInUse.md)
- [consumedBy](../properties/consumedBy.md)
- [forResource](../properties/forResource.md)
- [forState](../properties/forState.md)
- [hasAllocation](../properties/hasAllocation.md)
- [hasAsset](../properties/hasAsset.md)
- [hasAvailableCapacity](../properties/hasAvailableCapacity.md)
- [hasCapacity](../properties/hasCapacity.md)
- [hasOwner](../properties/hasOwner.md)
- [hasQuantity](../properties/hasQuantity.md)
- [hasResource](../properties/hasResource.md)
- [ownedBy](../properties/ownedBy.md)
- [participatesIn](../properties/participatesIn.md)
- [ResourceObjectProperty](../properties/ResourceObjectProperty.md)
- [resourceOf](../properties/resourceOf.md)
- [usedBy](../properties/usedBy.md)
