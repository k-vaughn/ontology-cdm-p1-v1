# Information technology - City data model - Part 1: Foundation level concepts - Mereology Pattern

This ontology specifies the foundation-level concepts related to the mereology pattern of the city data model.

This pattern imports the following files:

- [https://w3id.org/citydata/part1/v1/Core](https://w3id.org/citydata/part1/v1/Core)

This pattern does not contain any classes.
This module defines the following properties:

- [aggregateOf](../properties/aggregateOf.md)
- [aggregateOver](../properties/aggregateOver.md)
- [aggregationOf](../properties/aggregationOf.md)
- [componentOf](../properties/componentOf.md)
- [hasComponent](../properties/hasComponent.md)
- [hasProperPart](../properties/hasProperPart.md)
- [immediateComponentOf](../properties/immediateComponentOf.md)
- [MereologyObjectProperty](../properties/MereologyObjectProperty.md)
- [partOf](../properties/partOf.md)
- [properPartOf](../properties/properPartOf.md)


The formal definition of this pattern is available in TURTLE Syntax in two files, the [core semantics](../mereology-pattern.ttl) and the SHACL [restrictions](../mereology-shacl.ttl).
